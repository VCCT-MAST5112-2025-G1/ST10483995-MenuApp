# Christoffel Cuisine Menu App

A React Native (Expo) app that allows users to view, edit, and manage a restaurantâ€™s menu in an elegant and interactive interface.  
Built with **TypeScript**, **Expo**, and **React Native**, this app demonstrates dynamic data handling using **React Context** and **Hooks**.

---

## Features

-  **View Menu Categories:** Starters, Main, Desserts, and Specials.  
-  **Add New Dishes:** Create new menu items with name, description, and price.  
-  **Category Navigation:** Tap any category image to switch between menu sections.  
-  **Edit Mode:** Easily toggle editing on or off.  
-  **Persistent Context Support:** If integrated with a global context provider, dishes remain synced across the app.  
-  **Average Price Section:** Displays sample average prices for each category.  
-  **Dynamic Dish List:** View all dishes currently on the menu.  

---

## Tech Stack

- **React Native** (via [Expo](https://expo.dev/))
- **TypeScript**
- **React Context API**
- **Hooks** (`useState`, `useEffect`, `useContext`)
- **FlatList** for dynamic rendering
